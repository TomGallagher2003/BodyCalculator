# Bodybuilding Calculator Suite

A mobile-first React application featuring essential calculators for bodybuilding and fitness tracking.

## Features

- **TDEE/BMR Calculator** - Calculate your Total Daily Energy Expenditure using the Mifflin-St Jeor formula
- **Macro Splitter** - Get personalized protein, fat, and carb targets based on your goals (cut/bulk/recomp)
- **Navy Body Fat Calculator** - Estimate body fat percentage using the U.S. Navy method

## Tech Stack

- React 19 + Vite 7
- Tailwind CSS v4
- Lucide React icons
- Vitest for testing

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Run tests
npm run test

# Build for production
npm run build
```
